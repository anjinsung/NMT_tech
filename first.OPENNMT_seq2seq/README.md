# OpenNMT Ko->En 기술분야 번역

### 구성 환경 (Python 3.7 사용)
```
setup.py 파일 확인
```

### 사용 방법
```
python translate.py --model data/model/model.pt --src data/1-1.txt --output data/1-1.en.txt --gpu 0 --verbose
```
해당 명령어 바탕으로 translate_variable.py 작성 진행
